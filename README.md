# SJCL Command Line Interface (sjcl-cli)

## Summary

Export your passman vault data from command line.

## Requirements

* [NodeJS](https://nodejs.org/en/)

## Installation

```bash
   git clone https://github.com/RenonDis/passman-cli-export

   cd passman-cli-export

   sudo npm install -g
```

## Usage

Launch it:
```bash
    passman-cli-export
```
